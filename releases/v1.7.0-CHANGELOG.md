# Release v1.7.0 - Bookmark Manager Plugin

**Release Date:** April 7, 2026

## Features

### 🔍 Browser History Search
- Added support for searching browser visit history alongside bookmarks
- Results now display in two organized sections: **Bookmarks** and **History**
- Each result shows relevant metadata (URL, domain, visit count)

### 🎨 UI Optimization with shadcn Components
- Replaced custom components with shadcn/ui for consistency
- Implemented `InputGroup` component for cleaner search interface
- Enhanced accessibility and visual hierarchy
- Better responsive design across all screen sizes

### 🖱️ Enhanced Right-Click Menu System
- Consolidated all bookmark operations into context menus
- Quick actions on hover (rename, delete icons)
- Support for:
  - Open link in new tab
  - Copy link to clipboard
  - Rename bookmark/folder
  - Move to folder
  - Delete item

### 📦 Manifest Updates
- Added `history` permission for browser history API access
- Renamed extension to "书签搜索管理" (Bookmark Search Manager)
- Updated description to reflect new capabilities

### 🧹 Code Cleanup
- Removed unused search history hook
- Eliminated redundant inline action rows
- Simplified component structure
- Improved type safety with BrowserHistoryItem type

1. Extract the release package
2. For Chrome: Load unpacked extension from `dist_chrome/`
3. For Firefox: Use `about:debugging` and load `dist_firefox/manifest.json`

## Known Limitations

- Browser history search limited to 30 results per query
- History search requires browser history permission
