# Releases

This directory stores packaged release artifacts for local verification and manual uploads.

## Naming convention

- `bookmark-manager-chrome-vX.Y.Z.zip`: Chrome build package
- `bookmark-manager-firefox-vX.Y.Z.zip`: Firefox build package
- `bookmark-manager-source-vX.Y.Z.tar.gz`: Source code package
